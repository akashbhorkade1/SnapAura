# SnapAura – AI Photo Enhancer

SnapAura is a fast, free, and beautiful AI-powered app that restores and enhances your photos using Replicate's GFPGAN model.

## Features
- Upload or paste image URL
- AI enhancement via Replicate
- Side-by-side comparison
- Download enhanced photo
- Clean UI with grey theme and animation

## How to Use
1. Upload a `.jpg` or `.png` or paste a public image URL
2. Click enhance
3. View and download your enhanced photo

## Tech Stack
- Streamlit
- Replicate API
- Python (requests, Pillow, OpenCV)
